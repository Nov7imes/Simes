# Changelog

完整的历史开发记录随发布包提供，文件名为 `Simes-开发更新日志.md`。

## 1.4.0 — 2026-07-24

### Added

- 发酵桶与普通厨具信息卡。
- 烹饪时钟材料同步、服务器权威时间校准和本地平滑倒计时。
- 发酵投入、取出、开盖中断与重新校准诊断。
- 重新登录后通过准星重新发现发酵桶。

### Fixed

- 修复快速投入或取出时本地数量重复计算、遗漏或回滚错误。
- 修复发酵中断后继续使用旧倒计时。
- 修复服务器已确认完成但卡片仍显示“预计已到”。
- 修复展示实体偏移或短暂替换时，重登后无法重新发现发酵桶、钟查询丢失目标，以及服务器完成状态无法同步到信息卡。
- 修复厨具识别、开盖计时和未知 Boss Bar 被错误隐藏。
- 修复 Debug 实际十分钟上限与启动说明不一致。

### Changed

- 项目许可由 MIT 调整为 Simes Source-Available License 1.0；内置 AutoMessage 的 CC0-1.0 许可保持不变。

### Compatibility

- Minecraft 1.21.8
- Fabric Loader 0.17.2+
- Fabric API 0.136.1+1.21.8
- Java 21
